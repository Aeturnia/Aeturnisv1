## ðŸŽ® Example Usage
```typescript
// Client connection example
const socket = io('http://localhost:3001', {
  auth: { token: 'jwt-token-here' },
  reconnection: true,
  reconnectionAttempts: 5
});

// Listening for events

  console.log(`${data.sender}: ${data.message}`);
});

  x: 100,
  y: 200,
  zoneId: 'tavern-district'
});
```
